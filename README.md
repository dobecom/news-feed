## Overview

뉴스피드 백엔드 서비스 개발

## Installation

```bash
$ yarn
$ npx prisma generate
$ docker pull redis:latest
$ docker pull postgresql:13
$ docker compose up dev-db -d
$ docker compose up redis -d
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Test

준비중
```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
